AR Art Studio

現実世界をキャンバスに。

AR Art Studioは、スマートフォンのカメラやPCのWebカメラを使って、映し出された現実の風景にリアルタイムで「お絵描き」や「画像の合成」ができるWebアプリケーションです。

アプリのインストールは不要。ブラウザを開くだけで、あなたの目の前の景色がアート作品に変わります。

🎨 主な機能

1. リアルタイム・ARお絵描き

カメラの映像の上に、指やマウスを使って自由な線を描くことができます。

多彩な色: カラーピッカーで好きな色を選択可能。

ブラシ調整: 線の太さをスライダーで直感的に変更。

消しゴムモード: 描いた線を部分的に消すことも可能です。

2. 画像合成（オーバーレイ）

スマホ内の写真やイラスト画像を読み込み、カメラ映像に重ねることができます。

透明度調整: 画像を半透明にして、風景に幽霊のように溶け込ませることができます。

サイズ変更: 遠近感に合わせて画像の大きさを調整できます。

3. 思い出を保存（スナップショット）

「風景 ＋ お絵描き ＋ 合成画像」を組み合わせた画面を、ワンタップで画像データ（PNG形式）として保存できます。

📱 画面と使い方の説明

メイン画面（トップバー）

画面上部にあるアイコンです。

🔄 カメラ切り替え:
スマートフォンの「外側カメラ」と「自撮り用カメラ」を切り替えます。

🖼 画像読み込み:
合成したい写真やイラストを端末から選択してアップロードします。

画像設定パネル

画像を読み込むと右上に表示されます。

透明度スライダー: 左に動かすと薄く、右に動かすと濃くなります。

サイズスライダー: 画像の拡大・縮小ができます。

削除ボタン: 読み込んだ画像を消去します。

ツールバー（画面下部）

お絵描きの道具箱です。

🎨 色選択: 現在の色を表示。タップするとカラーピッカーが開きます。

━ ブラシサイズ: スライダーで筆の太さを調整します。

🖊 / ⌫ ブラシ・消しゴム切替: 描くモードと消すモードを切り替えます。

🗑 全消去: キャンバスに描いた絵をすべて消します（確認画面が出ます）。

📷 保存ボタン: 現在の画面を撮影し、画像としてダウンロードします。

💡 こんな使い方がおすすめ

街中に落書き: 実際の壁や道路には描けない落書きを、AR空間で楽しむ。

推しと撮影: 好きなキャラクターの画像を読み込み、現実の風景の中にいるように配置して記念撮影。

ファッションチェック: 自撮りモードにして、自分の服にバーチャルな模様やアクセサリーを描き足す。

メモ・記録: 撮影した商品の横に、値段や感想を直接書き込んで保存する。

⚠️ 注意事項・推奨環境

カメラの許可: 初回起動時にブラウザからカメラの使用許可を求められます。「許可」を選択してください。

ブラウザ: 最新の Google Chrome, Safari, Edge などで動作します。

プライバシー: 映像はすべてお使いの端末内で処理されます。サーバーに映像が送信されることはありませんので、安心してお使いください。
